位运算

按位与 &

在进行位运算的时候要把数据转换成二进制的补码形式（两位同为1结果为1），运算结束后再转换为10进制返回

应用：

  • n & 1 可以计算n的二进制n/2的余数（包括带符号数）

按位或 |

在进行位运算的时候要把数据转换成二进制的补码形式（两位只要有1位为1结果为1），运算结束后再转换为10进制返回

按位异或 ^

在进行位运算的时候要把数据转换成二进制的补码形式（两位相同为0，不同为1），运算结束后再转换为10进制返回

**应用\

  • 一般应用到加密（a两次异或b 结果还是a a ^ b ^ b == a）

  • 算法题：实现两个变量值的交换

按位取反 ~ 单目运算符

在进行位运算的时候要把数据转换成二进制的补码形式（0变1，1变0），运算结束后 再转换为10进制返回

位运算 << >> >>>

  • << 右边用0填充
    • n << i n * 2 的i次幂
  • >> 符号位是几就用几填充
    • n >> i n / 2 的i次幂
  • >>> 无符号右移———左边用0填充
    • n >> i n / 2 的i次幂
